数据类型

发表于 2019-09-20 分类于 js基础

数据类型

  1. 6种基本数据类型，一种复杂数据类型object
  2. 用typeof可以识别基本数据类型和function（null不可以，返回object，array返回object）
  3. instanceof 表示一个对象是否是另一个对象的实例

undefined

在使用var声明变量但未对其初始化时，这个变量就是undefined。

null

返回object，从逻辑角度来看，表示一个空对象指针

Boolean

“”(空字符串)、0、NaN、null、undefined通过Boolean()转换为的类型都为false。

number

  1. 保存浮点数需要的内存空间是保存整数值的两倍
  2. 可以用e表示法，默认情况下，会将小数点后面有6个零以上的转为e表示法保存
  3. isFinite()判断是否在Math.MIN_VALUE和Math.MAX_VALUE之间
  4. isNaN()和Number()在对象调用时会调用valueOf()方法，如果返回的值还是不能转换，调用toString()方法
  5. Number()会将false、null、空字符串转为0，true转为1，undefined转为NaN
  6. 一元加操作符相当于Number()
  7. parseInt()转换空字符串得到NaN，一个一个字符解析，parseInt('2014aa')得到2014，且可以识别十六进制符前置0x，也可传第二个参数表示使用的基数。

string

toString()参数可以传基数（几进制）

object

Object的每个实例都有以下属性和方法

  1. constructor
  2. hasOwnProperty(name)
  3. isPrototypeOf(object)
  4. propertyIsEnumrable(name)
  5. toLocalString()
  6. toString()
  7. valueOf()

操作符

关系操作符

  1. 如果两个都是数字，则数值比较
  2. 如果两个字符串，则比较字符编码值
  3. 如果一个是数值，则转换另一个为数值
  4. 如果一个是布尔，则转其为数值
  5. 如果一个是对象，则调用其valueof，不行再调用toString

+运算符

面试题：

null == undefined 为什么为true

null表示”没有对象”，即该处不应该有值。

（1） 作为函数的参数，表示该函数的参数不是对象。

（2） 作为对象原型链的终点。

undefined表示”缺少值”，就是此处应该有一个值，但是还没有定义。

（1）变量被声明了，但没有赋值时，就等于undefined。

（2) 调用函数时，应该提供的参数没有提供，该参数等于undefined。

（3）对象没有赋值的属性，该属性的值为undefined。

（4）函数没有返回值时，默认返回undefined。

ECMAScript认为undefined是从null派生出来的，所以把它们定义为相等的。
