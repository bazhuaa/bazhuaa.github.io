缓存

发表于 2019-09-19 更新于 2019-09-20 分类于 http

浏览器存在一个缓存数据库，用于储存一些不经常变化的静态文件（图片、css、js等）。

强制缓存

当缓存数据库中已有所请求的数据时。客户端直接从缓存数据库中获取数据。当缓存数据库中没有所请求的数据时，客户端的才会从服务端获取数据。

对于强制缓存，服务器响应的header中会用两个字段来表明——Expires和Cache-Control。

Symbol

发表于 2019-09-19 更新于 2019-09-20 分类于 js基础

Symbol代表一个独一无二的值，对象的属性可以是字符串，也可以是symbol，可以保证属性名不起冲突。

Symbol的参数只能是字符串，如果为对象，将会调用toString方法，只表示对当前symbol值的描述（可以用symbol.prototype.description取到），相同参数的返回值也是不同的.
